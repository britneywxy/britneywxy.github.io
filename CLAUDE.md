## Design rules
- Warm, light mode, personal and slightly playful
- Background: white or cream
- Accent colors: terracotta, warm coral, or dusty rose
- Google Fonts only
- Mobile responsive always
- Subtle cat reference somewhere (she has a cat)

## Structure
- index.html — entire site (HTML + embedded CSS)
- No build tools, no npm, no frameworks

## Sections (in order)
1. Hero — name, tagline, contact links
2. About Me
3. Experience — timeline style
4. Education
5. Skills — grouped by category
6. Projects — placeholder cards (Coming Soon)
7. Footer with contact

## Coding rules
- Never break mobile responsiveness
- Keep all CSS embedded in index.html
- When adding projects, follow existing card pattern
- Preserve warm color palette in all new additions
